Responsive web pages

What a responsive site is, and why responsiveness is important.

It is a site that responds to user's behaviour and environment based on the screen size and resolution. Responsive web page will adjust the contents, size, position or appearance of elements to the device it is viewed on. Responsive sites use Media Queries, flexible grids, layouts and images.

What mobile first design is, and why it's important.

Mobile first design is an approach based on assumptiuon that most of the website wisitors will be viewing the page on a smaller devices. This approach is based on creating websites for mobile screen-size first, the web site is later adjusted to larger screen size if the screen size is larger. It is important because it prevents mobiles from fetching large amounts of unneccessary data.

What frameworks are, and their pros and cons.

Frameworks are standardized sets of rules, concepts, and criteria for dealing with a common type of problem. There are two types of frameworks - backend and frontend. Backend frameworks deal with the logis and server-side. Frontend frameworks usually consist of a package made up of a structure of files and folders of standardized code (HTML, CSS, JS documents etc.)

What a wireframe is and why we use it.

Wireframe is a page schematic, a drawing or visual guide that represents the skeletal framework of a website. Wireframes show the website’s content, including interface elements and navigational elements, and how they work together.

We use wireframes to make the design and development process more clear and easy. It also allows to re-think the page's content, elements, purpose and behaviour.
