HTML CSS DOM

  • Can you think of an analogy to describe HTML, CSS and the DOM?

    • I am thinking about HTML elements as lego bricks or decoration elements in a house which are beautified by CSS and arranged in the browser thanks to the plan provided by DOM.

  • How would you describe it to your non-technical friend?

    • HTML elements are Lego bricks, CSS defines the bricks' colors and shapes, the browser uses lego elements to build contents on the page and knows how to do that thanks to DOM, which is kinda manual for the browser how to build a lego spaceship.

  • What is meant by boxifying design?

    • Boxifying is an approach to develop a website. Chopping up website elements to boxes and identifying relations between them.

  • What is the box model?

    • Box model refers to seeing website elements as boxes. We may give those boxes names, for examle by creating divs and adding classes to them in HTML document. Later, we can style, position and create relations between the boxes.
